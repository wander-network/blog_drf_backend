📄 博客系统 API 文档
基于 Django REST Framework 的博客后端 API

基础信息
项目	内容
Base URL	http://localhost:8000/api/
认证方式	JWT Token
请求格式	application/json
响应格式	application/json
认证说明
登录后获取 access_token，在请求头中添加：

text
Authorization: Bearer <your_access_token>
🔐 1. 认证与用户
1.1 用户注册
http
POST /api/register/
参数	类型	必填	说明
username	string	✅	用户名
password	string	✅	密码
email	string	✅	邮箱
请求示例：

json
{
    "username": "zhangsan",
    "password": "123456",
    "email": "zhangsan@example.com"
}
响应示例（201 Created）：

json
{
    "id": 1,
    "username": "zhangsan",
    "email": "zhangsan@example.com"
}
1.2 用户登录
http
POST /api/login/
参数	类型	必填	说明
username	string	✅	用户名
password	string	✅	密码
请求示例：

json
{
    "username": "zhangsan",
    "password": "123456"
}
响应示例（200 OK）：

json
{
    "access": "eyJhbGciOiJIUzI1NiIs...",
    "refresh": "eyJhbGciOiJIUzI1NiIs...",
    "user": {
        "id": 1,
        "username": "zhangsan",
        "email": "zhangsan@example.com"
    }
}
1.3 获取个人信息
http
GET /api/profile/
请求头： Authorization: Bearer <access_token>

响应示例（200 OK）：

json
{
    "id": 1,
    "username": "zhangsan",
    "email": "zhangsan@example.com"
}
1.4 刷新 Token
http
POST /api/token/refresh/
参数	类型	必填	说明
refresh	string	✅	刷新令牌
请求示例：

json
{
    "refresh": "eyJhbGciOiJIUzI1NiIs..."
}
响应示例（200 OK）：

json
{
    "access": "eyJhbGciOiJIUzI1NiIs..."
}
📝 2. 文章管理
2.1 获取文章列表
http
GET /api/posts/
参数	类型	说明
page	int	页码，默认 1
page_size	int	每页数量，默认 10
search	string	搜索标题和内容
category	int	按分类 ID 过滤
tag	int	按标签 ID 过滤
ordering	string	排序，如 -created_at
响应示例（200 OK）：

json
{
    "count": 25,
    "next": "http://localhost:8000/api/posts/?page=2",
    "previous": null,
    "results": [
        {
            "id": 1,
            "title": "Django REST Framework 入门",
            "summary": "本文介绍 DRF 的基本使用方法...",
            "content": "完整内容...",
            "author": {
                "id": 1,
                "username": "zhangsan"
            },
            "category": {
                "id": 1,
                "name": "Python"
            },
            "tags": [
                {"id": 1, "name": "Django"},
                {"id": 2, "name": "API"}

            "likes": 15,

2.3 创建文章
http
POST /api/posts/
请求头： Authorization: Bearer <access_token>

参数	类型	必填	说明
title	string	✅	标题
content	string	✅	内容
summary	string	❌	摘要（自动截取）
category	int	❌	分类 ID
tags	list	❌	标签 ID 列表
请求示例：

json
{
    "title": "Django REST Framework 入门",
    "content": "完整内容...",
    "category": 1,
    "tags": [1, 2]
}
响应示例（201 Created）：

json
{
    "id": 1,
    "title": "Django REST Framework 入门",
    "content": "完整内容...",
    "author": {
        "id": 1,
        "username": "zhangsan"
    },
    "category": {
        "id": 1,
        "name": "Python"
    },
    "tags": [
        {"id": 1, "name": "Django"},
        {"id": 2, "name": "API"}
    ],
    "views": 0,
    "likes": 0,
    "created_at": "2026-04-19T10:00:00Z"
}

💬 5. 评论管理
5.1 获取文章评论列表
http
GET /api/posts/{post_id}/comments/
响应示例（200 OK）：

json
[
    {
        "id": 1,

5.2 发表评论
http
POST /api/posts/{post_id}/comments/
请求头： Authorization: Bearer <access_token>

参数	类型	必填	说明
content	string	✅	评论内容
请求示例：

json
{
    "content": "这篇文章写得很好！"
}
响应示例（201 Created）：

json
{
    "id": 1,
    "author": {
        "id": 1,
        "username": "zhangsan"
    },
    "content": "这篇文章写得很好！",
    "created_at": "2026-04-19T10:30:00Z"
}
5.3 删除评论
http
DELETE /api/comments/{id}/
请求头： Authorization: Bearer <access_token>（必须是评论作者）

响应示例（204 No Content）：（无返回内容）

🔢 HTTP 状态码说明
状态码	说明
200	请求成功
201	创建成功
204	删除成功（无返回内容）
400	请求参数错误
401	未认证（未提供 Token）
403	无权限（Token 无效或不是作者）
404	资源不存在
500	服务器内部错误
